Adapter.remove method allows to remove items from current uiScroll buffer. This method accepts 1 argument which is a predicate function applying to each item in the current buffer. If the return value is true, the item will be removed. In this demo we have pretty simple predicate: 

{{predicateDescription}}

Here id is being received from the template as the id of the item the user ckicked on.

The very important thing is to syncronize our datasource with the changes we are making over the uiScroll buffer via Adapter.remove. Generally this is the App component responsibility, and in this demo it is being provided via doRemoveDatasource(id) method. It removes item with given id from the initial dataset, and decrements id of all items which ids are more than given one.
